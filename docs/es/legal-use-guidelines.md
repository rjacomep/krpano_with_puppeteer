## Propósito y Estado Legal

**Krpano QA Tester** es un marco legítimo de automatización y aseguramiento de calidad diseñado para probar tours panorámicos que posees o tienes autorización explícita para probar.

✅ **USOS LEGALES:**
- Pruebas QA de tus propios tours krpano
- Pruebas de tours para los que tienes permiso explícito
- Integración en pipelines CI/CD para QA automatizado
- Pruebas de regresión visual entre versiones
- Análisis de rendimiento y compatibilidad
- Validación de accesibilidad
- Documentación e informes internos

❌ **USOS ILEGALES/NO AUTORIZADOS:**
- Pruebas de tours sin autorización
- Eludir controles de acceso
- Redistribución no autorizada de contenido
- Raspado de contenido con derechos de autor
- Reventa comercial de activos extraídos
- Usar tours para análisis competitivo sin permiso

---

## Requisitos de Autorización

### Antes de Usar Esta Herramienta:

1. **Verifica Propiedad o Permiso**
   - Eres propietario del tour siendo probado, O
   - Tienes autorización escrita del propietario del tour

2. **Respeta los Términos de Servicio**
   - Verifica los términos de servicio del sitio web
   - Honra cualquier restricción de robots.txt
   - Sigue las directrices de limitación de velocidad

3. **Respeta la Propiedad Intelectual**
   - No redistribuyas contenido extraído
   - No uses contenido para propósitos comerciales sin permiso
   - Acredita a los creadores apropiadamente

4. **Usa Solo para Pruebas Internas**
   - Los resultados de pruebas son para tu equipo QA interno
   - No compartas contenido extraído públicamente
   - Mantén el ambiente de prueba privado

---

## Cumplimiento Técnico

### Prácticas de Prueba Respetuosa

Este marco incluye prácticas respetuosas integradas:

- **Limitación de Velocidad**: Los delays configurables evitan sobrecarga del servidor
- **Limitación de Tasa**: Backoff exponencial para fallos
- **Rotación de User Agent**: Se identifica como navegador
- **Control de Concurrencia**: Limita solicitudes paralelas

## Privacidad y Seguridad de Datos

### Lo Que Esta Herramienta Hace:
✅ Crea espejos locales del ambiente de prueba
✅ Genera reportes de capturas QA
✅ Registra métricas de rendimiento de pruebas

### Lo Que Esta Herramienta NO Hace:

## Política de Uso Aceptable

Los usuarios acuerdan:

1. **Autorización de Uso**
   - Solo prueba tours que posees o tienes permiso
   - Verifica autorización antes de probar
   - Mantén registros de autorización

2. **Operación Respetuosa**
   - Honra delays y límites configurados
   - No intentes eludir la limitación de velocidad
   - No hagas solicitudes excesivas

3. **Cumplimiento Legal**
   - Sigue leyes locales e internacionales
   - Respeta derechos de autor y propiedad intelectual
   - Honra términos de servicio del sitio web

4. **Divulgación Responsable**
   - Si encuentras problemas de seguridad, reporta responsablemente
   - No divulgues públicamente vulnerabilidades
   - Contacta al propietario directamente

5. **Sin Redistribución**
   - No compartas contenido extraído públicamente
   - No uses para propósitos comerciales sin permiso
   - Mantén ambientes de prueba privados

---

## Resumen: La Regla

**Usa esta herramienta SOLO para probar tours que posees o tienes permiso explícito para probar.**

Si no enviarías un correo al propietario del tour preguntando "¿Está bien si pruebo tu tour con esta herramienta?", no lo hagas.
